* The game itself runs inside the engine RPG 20XX, found at https://github.com/l33ner/RPG20XX-mirror

* 8-bit graphics pack was found at http://www.slimesalad.com/forum/viewgame.php?p=125406

* The font used is a modified version of Softball Gold, found at
  https://somepx.itch.io/humble-fonts-gold-ii

* Extracted graphics using OhrGfx, found at
  https://rpg.hamsterrepublic.com/ohrrpgce/Ohrgfx

* Split images using the Multi-Purpose Graphics Splitter, found at
  https://www.ascensiongamedev.com/files/file/16-multi-purpose-graphic-splitter/

* Combined images, and did a ridiculous amount of other general-purpose
  image manipulation, using IrfanView, found at https://www.irfanview.com/

* Created the world map using Tiled, found at https://thorbjorn.itch.io/tiled

* Decided upon theme (Murder Mystery) and setting (Dark Fantasy), using the Story Theme
  and Setting Generator, found at https://letsmakeagame.net/story-theme-and-setting-generator/

* Made image backgrounds transparent for all manner of object sprites using the
  Transparent Background Tool, found at https://www12.lunapic.com/editor/?action=transparent

* The fictional name used in the game title, Icsmaun, is derived from an output of the
  Random Letter Generator (namely “Icksmawn”), while the names of the protagonists are
  unmodified outputs of it. This can be found at https://randomwordgenerator.com/letter.php
